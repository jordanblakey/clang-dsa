### C Related Unix Commands

- `gcc` Compiler
- `nm`	Name List	Shows the symbols (functions/variables) in a binary.
- `ar`	Archiver	Used to create static libraries (.a files).
- `ld`	Loader	The actual linker that combines .o files into an app.
- `cc`	C Compiler	The original command for the C compiler (now often a link to gcc).
- `as`	Assembler	Turns assembly code into machine code.


### C Related File Extensions

- `.c` Source. Human-readable code and logic. The raw blueprints.
- `.h` Header. The function "menu" or signatures.	The table of contents.
- `.o` Object. Binary version of a single .c file. A pre-built wall.
- `.a` Static Archive. A bundle of .o files pasted into your app. A house kit you buy and own.
- `.so`	Shared Object	A library file linked at runtime.	A tool you rent when needed.
- `.d` Dependency.	A map of which headers affect which code.	A "Notice of Change" list.
